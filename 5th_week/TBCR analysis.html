Data

All the TRUST output data that we will use in this class can be found at /lustre1/share/airr/data

In this folder, there are two subfolders, TRUST_output and OV_results. TRUST_output folder storages output files of one sample from TCGA RNA-seq data processed by TRUST3 tool. OV_results folder storages _report.tsv files extracted by TRUST3 in TCGA-OV cohort (375 samples).

cd /lustre1/share/airr/data/TRUST_output
ls

TRUST_output
    ├── *_anno.fa  ## fasta format for the annotation of the consensus assembly
    ├── *_assembled_reads.fa  ## complete read sequence of the assemblies
    ├── *_cdr3.out  ## the CDR1,2,3 and gene information for each consensus assemblies
    ├── *_cluster.out  ## cluster information of consensus assemblies
    ├── *_evo.out  ## evolution information of final TCR/BCR sequences
    ├── *_final.out  ## final contigs and corresponding nucleotide weight
    ├── *_full.fa  ## complete TCR/BCR sequences file
    ├── *_raw.out   ## raw contigs and corresponding nucleotide weight
    ├── *_report.tsv  ## report file focusing on CDR3 and is compatible with other repertoire analysis tool such as VDJTools.
    └── *_simplerep.out  ## simplified report file

The raw RNA-seq data is from TCGA public dataset.

Part 4. Calculate TCR/BCR diversity index of each sample

Before we discuss TCR/BCR diversity, we should define some indices to quantify the diversity of AIRR.
It has been introduced these indices to describe one's AIRR diversity (from Chiffelle et al.):

  • raw diversity: Number of unique CDR3 sequence.
  • clonotypes per thousand kilo reads (CPK): (Unique CDR3 count / Total reads) * 1000.
  • entropy:
    H(X)=−∑i=1Npilog⁡piH(X) = -\sum_{i=1}^{N} p_i \log p_iH(X)=−i=1∑N​pi​logpi​
    where pip_ipi​ is the frequency of sequence iii in the repertoire and NNN is the total number of unique sequences.
  • clonality: measures the dominance of clones in a repertoire.
    Clonality=1−H(X)logNClonality = 1-\frac{H(X)}{log N}Clonality=1−logNH(X)​
  • Gini-Simpson index: the probability of two entities being chosen randomly to not belong to the same species.
    Gini−Simpsonindex=1−∑i=1Npi2Gini-Simpson index = 1-\sum_{i=1}^{N} p_i^2Gini−Simpsonindex=1−i=1∑N​pi2​

After knowing those indices, we can calculate them of each sample.

Note: Due to the heavy chain determines the binding capacity of BCR and beta chain for TCR, we only focus the heavy/beta chain diversity in the practice.

A. Test on previous sample

First, load the CDR3 sequence information.

air_df <- read.table("/lustre1/share/airr/data/TRUST_output/TCGA-02-0055-01A-01R-1849-01_report.tsv", sep='\t')
colnames(air_df) <- c("count","frequency","CDR3nt","CDR3aa","V","D","J","C")
print(head(air_df))

Then, extract rows of heavy chain of BCR.

IGH_df <- air_df %>%
  filter(grepl("^IGHV", V) | 
         grepl("^IGHD", D) | 
         grepl("^IGHJ", J) | 
         grepl("^IGHC", C))

print(head(IGH_df))

Now, we can calculate those indices:

  • raw diversity:
diversity <- nrow(IGH_df)
print(diversity)
  • CPK:
CPK <- diversity/sum(IGH_df$count) *1000
print(CPK)
  • entropy:
freqs <- IGH_df$count/sum(IGH_df$count)
entropy <- -sum(freqs * log2(freqs))
print(entropy)
  • clonality:
clonality <- 1 - (entropy/log2(diversity))
print(clonality)
  • gini-simpson index:
gini <- 1 - sum(freqs * freqs)
print(gini)

B. Test on cohort samples

There is TCGA-OV cohort data under OV_results folder. We can calculate indices for each of the samples.

## create a data.frame to storage the indices value of each sample
bcr_metrics_df <- data.frame()

## list the files under OV_results folder
files <- list.files("/lustre1/share/airr/data/OV_results")

## Following the code above, the indices for each sample are
## calculated in a loop and stored in bcr_metrics_df
for (file in files) {
  air_df <- read.table(paste0("/lustre1/share/airr/data/OV_results/",file), sep="\t")
  colnames(air_df) <- c("count","frequency","CDR3nt","CDR3aa","V","D","J","C")
  IGH_df <- air_df %>%
  filter(grepl("^IGHV", V) | 
         grepl("^IGHD", D) | 
         grepl("^IGHJ", J) | 
         grepl("^IGHC", C))

  diversity <- nrow(IGH_df)
  CPK <- diversity/sum(IGH_df$count) *1000

  freqs <- IGH_df$count/sum(IGH_df$count)
  entropy <- -sum(freqs * log2(freqs))

  clonality <- 1 - (entropy/log2(diversity))
  gini <- 1 - sum(freqs * freqs)
  
  bcr_metrics_row <- data.frame(diversity, CPK, entropy, clonality, gini)
  rownames(bcr_metrics_row) <- tools::file_path_sans_ext(basename(file))  # remove the file suffix

  bcr_metrics_df <- rbind(bcr_metrics_df, bcr_metrics_row)
}

print(head(bcr_metrics_df))
write.csv(bcr_metrics_df, "OV_bcr_diversity.csv")
